# VibesFlow

**Version:** 0.1.0

A cutting-edge React Native mobile application that transforms emotional experiences into personalized musical compositions using advanced AI and blockchain technologies.

## Key Features

- **Emotional Music Creation**: Transform your feelings into unique musical compositions
- **Blockchain Integration**: Secure music creation tracking on Sui Blockchain
- **AI-Powered Analysis**: Advanced mood sensing and processing algorithms
- **Cross-Platform**: Built with React Native for iOS, Android, and Web

## Technologies

- React Native for cross-platform mobile development
- Expo for rapid development and deployment
- Sui Blockchain for secure music creation tracking
- TensorFlow.js for AI-powered mood analysis
- Tone.js for dynamic musical experience rendering
- TypeScript for type safety

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   expo start
   ```

3. Scan the QR code with Expo Go app on your mobile device or press 'w' to open in web browser
